# VACANT-CLASSROOM-FINDER
A web-based Classroom Availability Management System built with HTML, CSS, PHP, and MySQL. Users can check and manage available classrooms by selecting time slots, making scheduling and room allocation easier for schools, colleges, and universities.

## Features

- View classroom availability by **time slot**.
- User-friendly **timestamp selection** for checking availability.
- Responsive design for desktops and tablets.
- Simple and clean interface using CSS.
- Easy integration with MySQL database.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL
